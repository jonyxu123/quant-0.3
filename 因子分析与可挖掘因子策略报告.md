#### 3. 因子计算逻辑评估（优化后）

**✅ 正确实现的核心因子**:
- **Beta计算**: 使用CAPM模型，带截距项的回归 `alpha = np.mean(stock_r) - beta * np.mean(idx_r)`，残差计算正确
- **IVOL计算**: 残差标准差计算正确 `residual = stock_r - alpha - beta * idx_r`
- **Altman-Z**: X1-X5公式正确，X4使用市值/总负债（标准上市公司版）
- **动量因子**: 多周期动量（3M/6M/12M）计算正确，相对强度减去指数收益
- **资金流因子**: 主力净流入、超大单/大单净流入计算正确
- **筹码因子**: 成本集中度、获利盘比例计算正确
- **MACD金叉**: 使用前一日数据判断真正金叉（前日DIF<=DEA，当日DIF>DEA）
- **Roll价差**: 使用协方差计算隐含价差 `2*sqrt(max(-cov(r_t, r_{t-1}), 0))`
- **Amihud非流动性**: `mean(|pct_chg| / amount) × 10^6` 计算正确

**⚠️ 需要优化的因子**:
- **V_NCPS**: 单位换算已修复 `eq / (total_share * 10000)`
- **EV/EBITDA**: 用EBIT+折旧摊销近似EBITDA，误差可接受但可改进
- **STUB因子**: 约15个因子返回0或NaN（融资、龙虎榜、事件等）

## 第二部分：因子计算合理性详细分析

### 一、波动率因子模块 (`volatility_factors.py`)

#### 1. Beta计算（已修复）

**原问题**: Beta = Cov(stock, idx) / Var(idx)，但代码中 `predicted = beta * idx_r` 计算的是预期收益，不是残差。正确公式应为 `residual = stock_r - alpha - beta * idx_r`，缺少alpha截距项。

**修复方案**:
```python
# 修复前
predicted = s['beta'] * idx_r
residual = stock_r - predicted

# 修复后
alpha = np.mean(stock_r) - beta * np.mean(idx_r)
residual = stock_r - alpha - beta * idx_r
s['ivol'] = np.std(residual)
```
